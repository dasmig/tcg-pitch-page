# TCG Project — Pitch Page

A single-page pitch site for **TCG Project**, the first AI-generated competitive trading card game.

## Quick Start

Open `index.html` in any browser. No build step required — everything is self-contained (HTML + embedded CSS + JS).

## Deploy

This page is designed to be deployed to GitHub Pages:

```bash
# Enable Pages in repo settings → Source: deploy from branch → main / root
```

Or any static hosting (Netlify, Vercel, Cloudflare Pages).

## What's Inside

- **Hero** — Elevator pitch with key metrics
- **Market Position** — Competitive comparison table vs MTG, Balatro, Marvel Snap
- **Generation Pipeline** — 4-stage AI process visualization
- **Scoring Engine** — Interactive demo of the Chips × Mult × xMult formula
- **PvP Design** — 3-lane simultaneous turn system
- **Architecture** — Tech stack and infrastructure overview
- **Traction** — Current build progress and metrics
- **Roadmap** — G1 through G5 milestones
- **Business Model** — Ethical F2P monetization strategy
- **Call to Action** — Contributors, investors, publishers

## Tech

- Pure HTML/CSS/JS (zero dependencies)
- Google Fonts: Cinzel, Inter, JetBrains Mono
- Intersection Observer for scroll animations
- Fully responsive (mobile → desktop)
